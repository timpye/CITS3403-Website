Central Processing Unit (CPU)

Description

The CPU stands for Central Processing Unit. Being the "brain" of the computer, the CPU functions as the main control center of all computer operations. It has four specific functions: to fetch, execute, decode, and store instructions. A CPU contains cores, which are units in a CPU that can run different processes. The more cores a CPU has, the faster the computer will be.

CPU Installation

You need:

  • A CPU
  • A motherboard with a compatible chipset (see Motherboard)
  • Thermal Paste
  • A CPU cooler (normally comes with the CPU)

    • Steps:
  • Holding the CPU by its sides, line up your CPU with the CPU socket on the motherboard.
  • Set the CPU into its socket; assuming it is correctly lined up, the CPU should drop right into place.
  • Push down the retention arm on the CPU socket.
  • Apply a layer of thermal paste to the top of the CPU lid.
  • Install the CPU cooler by lining up the screws with the holes in the motherboard, then lower the cooler onto the CPU and attach.

CPU Recommendations

The two main manufacturers for CPUs are Intel and AMD, both of which produce high-quality CPUs. For uses such as gaming, the AMD Ryzen and Intel Core lines are sufficient. Regarding uses such as workstations, both manufacture high-end desktop or workstation CPUs, such as Intel Xeon and AMD Ryzen Threadripper Pro. AMD are currently producing the Ryzen 5000 series and Intel are currently producing their 11th generation architecture, the Tiger Lake.
