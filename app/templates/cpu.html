Central Processing Unit (CPU)

Description

The CPU stands for Central Processing Unit. Being the "brain" of the computer, the CPU functions as the main control center of all computer operations. It has four specific functions: to fetch, execute, decode, and store instructions. A CPU contains cores, which are units in a CPU that can run different processes. The more cores a CPU has, the faster the computer will be.

CPU Installation

You need:

  • A CPU
  • A motherboard with a compatible chipset (see Motherboard)
  • Thermal Paste
  • A CPU cooler (normally comes with the CPU)

      • The first few steps of installing a CPU varies depending on the CPU brand. For an Intel CPU, the CPU retention bracket must be released through a small metal lever which is holding it down on the motherboard. For an AMD CPU, the CPU locking lever must be lifted up from the socket for the CPU to be placed.
